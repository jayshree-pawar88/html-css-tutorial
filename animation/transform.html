Assignment 1: Basic Transformations In this assignment, you will create a webpage with various HTML elements like images or divs. Then, you'll apply different transformations to these elements using the transform property in CSS. The transform property allows you to alter the appearance of elements in 2D or 3D space. You can rotate, scale, skew, and translate elements. For example, you can rotate an image by a certain angle, scale it to make it larger or smaller, skew it to give it a slanted appearance, and translate it to move it around the page.
